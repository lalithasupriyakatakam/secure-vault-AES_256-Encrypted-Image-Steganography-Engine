# secure-vault-AES_256-Encrypted-Image-Steganography-Engine
SecureVault: Multi-layer security system in C++ combining SHA-256 hashing, ECC key generation (GF-5), AES-256-CBC encryption, and randomized LSB steganography to hide encrypted messages inside images. Built from scratch with zero external libraries — achieves imperceptible ±1 pixel distortion and 100% verified message integrity.
